# kotlin-armeria-grpc(wiht.spring boot)
kotlin-grpc using spring boot + armeria

### Requirements ###

* Spring Boot 3.2.0
* Kotlin 1.9.20
* Kotlin Coroutines
* Java 18 (feat.corretto-18)
* Gradle 7.4.2

### Building the artifact ###

```
gradle build
```

### Running the application from command line ###

```
gradle bootRun
````
